# Lace


## Usage

### Take Screen Shot
```bash
lace shot
```

### Record Screen
```bash
lace rec
```

If you want specify output path (or name), you can use `--path` option.
```bash
# Specify output directory.
lace rec -p ~/Desktop

# Specify output directory and name.
lace rec -p ~/Desktop/new.png
```

### Installation

```bash
mint install mh-idea/lace
```
